This project is Command-Line interface model of an online retail platform.
A user can use the terminal to access the store. It allows a user to login as a seller or login as a buyer
since it's a thrift store. A user can login using there phone number and do the following
    AS A BUYER:
        1. buy an item
        2. add product to there cart
        3. checkout from there cart
        4. View there cart
    AS A SELLER: 
        1. Upload a new product
        2. View listed Items

It also gives the admin many functions like customer analysis and viewing listed products and active users.
